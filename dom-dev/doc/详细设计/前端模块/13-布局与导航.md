## 4. 页面状态（各 `views/*.vue`）

| 状态 | Element Plus |
|------|----------------|
| loading | `v-loading` / `el-table :loading` |
| empty | `el-empty` + CTA 按钮 |
| error | `el-alert` + 再読み込み |
| 操作成功 | `ElMessage.success` |
| 操作失败 | `ElMessage.error(apiMessage)` |

---

## 5. CRUD 交互约定

- **一覧页**：筛选 + 表格 + 分页；新增/编辑用 **el-dialog**（Users、Employees、Dorms 等）
- **复杂向导**：独立路由（入居 `el-steps`、Import `el-steps`）
- **详情页**：寮详情用 Tab（`el-tabs`）：基本情報 / 部屋 / 履歴
