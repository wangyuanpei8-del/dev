## 4. Axios 封装（`src/utils/request.js`）

```javascript
import axios from 'axios';
import { ElMessage } from 'element-plus';
import { useUserStore } from '@/store/user';
import router from '@/router';

const service = axios.create({
  baseURL: import.meta.env.VITE_API_BASE_URL,
  timeout: 30000,
});

let refreshPromise = null;

service.interceptors.request.use((config) => {
  const userStore = useUserStore();
  if (userStore.accessToken) {
    config.headers.Authorization = `Bearer ${userStore.accessToken}`;
  }
  return config;
});

service.interceptors.response.use(
  (response) => {
    const { code, message, data } = response.data;
    if (code !== 0) {
      ElMessage.error(message || 'エラーが発生しました');
      return Promise.reject({ code, message });
    }
    return data;
  },
  async (error) => {
    if (error.response?.status === 401) {
      const userStore = useUserStore();
      if (!refreshPromise) {
        refreshPromise = userStore.refreshToken().finally(() => { refreshPromise = null; });
      }
      const ok = await refreshPromise;
      if (ok) return service(error.config);
      userStore.logout();
      router.push('/login');
    }
    ElMessage.error(error.response?.data?.message || 'ネットワークエラー');
    return Promise.reject(error);
  },
);

export default service;
```

## 6. 权限

### 6.1 路由守卫

```javascript
router.beforeEach((to, from, next) => {
  const userStore = useUserStore();
  if (to.meta.public) return next();
  if (!userStore.accessToken) return next('/login');
  if (to.meta.permission && !userStore.hasPermission(to.meta.permission)) {
    return next('/403');
  }
  next();
});
```

### 6.2 指令 `v-permission`

```vue
<el-button v-permission="'occupancy:write'" type="primary">入居登録</el-button>
<el-button v-permission:role="'SYSTEM_ADMIN'">初回入寮日修正</el-button>
```

## 8. 页面开发标准（对齐智能体）

| 场景 | 实现 |
|------|------|
| 布局 | `layout/index.vue`：左菜单 + 顶栏 + `<router-view>` |
| 一覧 | 筛选 `el-form` + `el-table`（序号、多选、操作列）+ `el-pagination` |
| 新增/编辑 | **el-dialog** + `el-form` + rules（非独立 /new 页，复杂向导除外） |
| 删除/确定 | `ElMessageBox.confirm` |
| 日期 | `el-date-picker` type="date"，格式 `YYYY-MM-DD` |
| 消息 | `ElMessage` / `ElMessageBox` |
| 加载 | `v-loading` 或表格 `:loading` |

## 10. 数据刷新约定

| 模式 | 说明 |
|------|------|
| 一覧页 | `fetchList()` 方法；操作成功后重新调用 |
| Pinia | 跨页共享状态（用户信息、字典） |
| Composable | `useEmployeeList()` 封装 loading/list/total/page |

| 操作后 refresh | 模块 |
|----------------|------|
| 入居登记 | employees, occupancy, vacancies, dorms, dashboard |
| 寮費確定 | dorm-fees, dashboard |
| Import 成功 | employees, dorms, occupancy, dorm-fees |

## 14. 设计原则

1. 业务规则服务端强制执行；前端 rules 为辅助
2. 写操作 `v-permission`；Viewer 不显示按钮
3. API 路径以 `docs/stage-5-api-spec.md` 为准
4. 仅 PC 桌面端，不做移动端适配
